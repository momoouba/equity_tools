# 格隆汇网站正文提取优化说明

## 问题描述
格隆汇网站（https://www.gelonghui.com）的新闻页面使用 `article class="main-news article-with-html"` 标签来标记正文内容，但在AI重新分析后，没有正确提取正文。

## 优化内容

### 1. 优化匹配规则

**位置**：`news/server/utils/newsAnalysis.js:112-124`

**优化内容**：
- 添加了精确匹配 `article class="main-news article-with-html"` 的规则
- 支持单引号和双引号的class属性
- 使用单词边界 `\b` 确保精确匹配
- 处理Vue.js的特殊属性（如 `data-v-*`）

### 2. 优化匹配逻辑

**位置**：`news/server/utils/newsAnalysis.js:147-230`

**优化内容**：
- **全局匹配**：使用全局匹配查找所有可能的article标签，然后选择内容最长的
- **降低阈值**：对于 `main-news` 匹配规则，内容长度阈值从200字符降低到100字符，如果内容在50-100字符之间也尝试使用
- **详细日志**：添加了详细的调试日志，包括：
  - HTML检查（是否包含article标签、main-news、article-with-html）
  - 所有article标签的class属性
  - 每个匹配规则的尝试结果
  - 匹配到的article标签的完整信息

### 3. 匹配规则优先级

1. **最高优先级**：精确匹配 `class="main-news article-with-html"` 或 `class="article-with-html main-news"`
2. **次高优先级**：匹配包含 `main-news` 的 article 标签
3. **第三优先级**：匹配包含 `article-with-html` 的 article 标签
4. **其他规则**：继续尝试其他常见的正文容器

### 4. 内容长度要求

- **企查查、格隆汇等网站的main-news匹配规则（前4个规则）**：
  - 理想情况：≥100字符
  - 最低要求：≥50字符（如果内容在50-100字符之间，也会尝试使用）
- **其他匹配规则**：≥200字符

## 测试方法

### 1. 测试特定URL

可以手动测试格隆汇网站的URL：

```javascript
const newsAnalysis = require('./server/utils/newsAnalysis');
const url = 'https://www.gelonghui.com/news/5136685';

newsAnalysis.fetchContentFromUrl(url).then(content => {
  console.log('提取的正文长度:', content.length);
  console.log('正文预览（前500字符）:', content.substring(0, 500));
}).catch(error => {
  console.error('提取失败:', error);
});
```

### 3. 检查数据库中的提取结果

```sql
-- 查询格隆汇网站的新闻，检查正文提取情况
SELECT id, title, 
       LENGTH(content) as content_length,
       source_url,
       CASE 
         WHEN content IS NULL OR content = '' THEN '未提取'
         WHEN LENGTH(content) < 100 THEN '提取内容太短'
         ELSE '已提取'
       END as extract_status
FROM news_detail
WHERE source_url LIKE '%gelonghui.com%'
  AND created_at >= DATE('2025-12-18')
  AND delete_mark = 0
ORDER BY created_at DESC
LIMIT 20;
```

## 如果仍然无法提取

### 排查步骤

1. **查看日志**：检查是否有匹配规则的日志输出
2. **检查HTML结构**：确认实际的HTML结构是否符合预期
3. **检查内容长度**：确认提取的内容是否满足最小长度要求
4. **检查正则表达式**：如果HTML结构特殊，可能需要调整正则表达式

### 可能的问题

1. **HTML结构变化**：如果格隆汇网站更新了HTML结构，可能需要调整匹配规则
2. **嵌套article标签**：如果HTML中有嵌套的article标签，可能需要处理嵌套情况
3. **动态内容**：如果内容是动态加载的，可能需要等待JavaScript执行
