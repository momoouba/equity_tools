# Excel导出格式优化说明

## 📋 优化概述

对舆情信息导出的Excel文件进行了格式优化，提升了数据的可读性和专业性，包括时间字段格式化、超链接设置和表格样式美化。

## 🎯 主要优化

### 1. 时间字段格式化

#### 优化前
- 时间显示为文本格式
- 格式不统一，难以排序和筛选

#### 优化后
- **发布时间**和**创建时间**设置为Excel日期类型
- 统一格式：`yyyy-mm-dd hh:mm:ss`
- 支持Excel的日期排序和筛选功能
- 居中对齐显示

```javascript
// 时间字段处理
'发布时间': item['发布时间'] ? new Date(item['发布时间']) : null,
'创建时间': new Date(item['创建时间']),

// 设置Excel日期格式
ws[timeCell].t = 'd'; // 日期类型
ws[timeCell].z = 'yyyy-mm-dd hh:mm:ss'; // 日期格式
```

### 2. 超链接设置

#### 优化前
- 文章链接显示为普通文本
- 需要复制粘贴才能访问

#### 优化后
- **原文链接**设置为可点击的超链接
- 蓝色字体 + 下划线样式
- 鼠标悬浮显示提示信息
- 直接点击即可打开链接

```javascript
// 超链接设置
ws[linkCell].l = { 
  Target: ws[linkCell].v, 
  Tooltip: '点击打开链接' 
};
ws[linkCell].s.font = { 
  color: { rgb: "0000FF" }, 
  underline: true 
};
```

### 3. 表格样式美化

#### 表头样式
- **背景色**：蓝色 (#4472C4)
- **字体**：白色加粗
- **对齐**：居中对齐
- **边框**：黑色细边框

#### 数据行样式
- **边框**：浅灰色细边框 (#CCCCCC)
- **对齐**：左对齐，顶部对齐
- **自动换行**：长文本自动换行
- **企业名称**：加粗显示，深灰色字体

#### 功能增强
- **冻结窗格**：冻结表头行，滚动时表头始终可见
- **自动筛选**：启用Excel自动筛选功能
- **列宽优化**：根据内容调整合适的列宽

## 🔧 技术实现

### 1. 日期格式处理

```javascript
// 将字符串日期转换为Date对象
const formattedData = data.map(item => ({
  ...item,
  '发布时间': item['发布时间'] ? new Date(item['发布时间']) : null,
  '创建时间': new Date(item['创建时间'])
}));

// 设置Excel单元格为日期类型
ws[timeCell].t = 'd';
ws[timeCell].z = 'yyyy-mm-dd hh:mm:ss';
```

### 2. 超链接实现

```javascript
// 检查是否为有效链接
if (ws[linkCell].v && ws[linkCell].v.startsWith('http')) {
  // 设置超链接属性
  ws[linkCell].l = { 
    Target: ws[linkCell].v, 
    Tooltip: '点击打开链接' 
  };
  
  // 设置超链接样式
  ws[linkCell].s.font = { 
    color: { rgb: "0000FF" }, 
    underline: true 
  };
}
```

### 3. 表格样式设置

```javascript
// 表头样式
ws[headerCell].s = {
  font: { bold: true, color: { rgb: "FFFFFF" } },
  fill: { fgColor: { rgb: "4472C4" } },
  alignment: { horizontal: "center", vertical: "center" },
  border: { /* 边框设置 */ }
};

// 数据行样式
ws[cellRef].s = {
  border: { /* 浅色边框 */ },
  alignment: { 
    horizontal: "left", 
    vertical: "top",
    wrapText: true 
  }
};
```

### 4. 功能增强

```javascript
// 冻结表头
ws['!freeze'] = { xSplit: 0, ySplit: 1 };

// 自动筛选
ws['!autofilter'] = { ref: ws['!ref'] };

// 列宽设置
ws['!cols'] = colWidths;
```
