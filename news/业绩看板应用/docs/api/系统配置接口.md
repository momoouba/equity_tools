## 3. 获取SQL配置列表

### 基本信息

- **接口地址**: `GET /api/performance/config/sql`
- **接口说明**: 获取数据接口SQL配置列表

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| page | number | 否 | 页码，默认 1 |
| pageSize | number | 否 | 每页条数，默认 20 |
| databaseName | string | 否 | 按数据库名称筛选 |

### 响应数据

```typescript
interface SQLConfigItem {
  id: string;                // 配置ID
  databaseName: string;      // 数据库名称
  interfaceName: string;     // 接口名称
  execOrder: number;         // 执行顺序
  createTime: string;        // 创建时间
  creator: string;           // 创建人
  updateTime: string;        // 修改时间
  updater: string;           // 修改人
}

interface SQLConfigListResponse {
  list: SQLConfigItem[];
  page: number;
  pageSize: number;
  total: number;
}
```

### SQL参考

```sql
SELECT 
  F_Id as id,
  database_name,
  interface_name,
  exec_order,
  F_CreatorTime as createTime,
  F_CreatorUserId as creator,
  F_LastModifyTime as updateTime,
  F_LastModifyUserId as updater
FROM b_sql
WHERE F_DeleteMark = 0
ORDER BY exec_order ASC
```

## 5. 创建SQL配置

### 基本信息

- **接口地址**: `POST /api/performance/config/sql`
- **接口说明**: 创建新的SQL配置

### 请求参数

```typescript
interface CreateSQLConfigRequest {
  databaseName: string;      // 数据库名称
  interfaceName: string;     // 接口名称
  sqlContent: string;        // SQL内容
  execOrder: number;         // 执行顺序
}
```

### 请求示例

```json
{
  "databaseName": "业务数据库",
  "interfaceName": "管理规模数据查询",
  "sqlContent": "SELECT * FROM manage_data WHERE date <= ?",
  "execOrder": 1
}
```

### 验证规则

1. `interfaceName` 不能为空，长度 1-100 字符
2. `sqlContent` 不能为空，长度 1-10000 字符
3. `execOrder` 必须为正整数
4. SQL 内容禁止包含危险操作：DROP、DELETE、UPDATE、INSERT、TRUNCATE、ALTER

### 错误码

| 错误码 | 说明 |
|--------|------|
| 400 | 参数验证失败 |
| 400 | SQL包含危险操作 |
| 409 | 执行顺序已存在 |

## 8. 测试SQL配置

### 基本信息

- **接口地址**: `POST /api/performance/config/sql/:id/test`
- **接口说明**: 测试SQL配置是否正确

### 请求参数

```typescript
interface TestSQLRequest {
  testDate: string;          // 测试日期，格式 YYYY-MM-DD
  limit?: number;            // 返回条数限制，默认 100，最大 1000
}
```

### 请求示例

```json
{
  "testDate": "2026-02-28",
  "limit": 10
}
```

### 响应数据

```typescript
interface TestSQLResponse {
  success: boolean;
  executionTime: number;     // 执行时间（毫秒）
  rowCount: number;          // 返回行数
  columns: string[];         // 列名列表
  preview: any[];            // 预览数据（前N条）
  error?: string;            // 错误信息（执行失败时）
}
```

### 响应示例

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "success": true,
    "executionTime": 1250,
    "rowCount": 156,
    "columns": ["fund", "sub_amount", "paid_in_amount"],
    "preview": [
      {"fund": "国方一期", "sub_amount": 100000000, "paid_in_amount": 50000000},
      {"fund": "国方二期", "sub_amount": 200000000, "paid_in_amount": 100000000}
    ]
  },
  "timestamp": 1704067200000
}
```

### 安全说明

1. 测试SQL在只读事务中执行
2. 自动添加 LIMIT 限制，防止大数据量查询
3. 超时时间为 30 秒
4. 只返回前 N 条数据预览，不返回全部数据
