# 业绩看板 API 接口文档

## 文档说明

本文档定义了业绩看板系统的所有 RESTful API 接口规范，供前后端开发人员参考使用。

6. [分享接口](./分享接口.md)

## 接口列表概览

| 模块 | 接口数量 | 主要功能 |
|------|----------|----------|
| 版本管理 | 6 | 版本查询、创建、锁定、删除 |
| 数据查询 | 12 | 各模块数据查询、弹窗数据 |
| 数据导出 | 8 | Excel导出功能 |
| 系统配置 | 5 | SQL配置、指标说明配置 |
| 分享 | 2 | 分享链接生成、验证 |

## 基础信息

- **Base URL**: `/api/performance`
- **协议**: HTTPS
- **数据格式**: JSON
- **字符编码**: UTF-8
- **时区**: Asia/Shanghai (UTC+8)

## 认证方式

所有接口（除登录相关）需在请求头中携带 JWT Token：

```http
Authorization: Bearer <token>
```
