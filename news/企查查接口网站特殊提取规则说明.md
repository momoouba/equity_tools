# 企查查接口网站特殊提取规则说明

## 📋 功能说明

为企查查接口的不同新闻网站添加了特殊的正文提取规则，确保能够正确提取正文内容。

## 🎯 支持的网站和提取规则

### 1. 中国民用航空网
- **URL匹配**：`caacnews.com.cn` 或 `caacnews.com`
- **提取规则**：从 `<div class="entry-content">` 中提取正文
- **实现位置**：`extractArticleContent` 方法

### 2. 长江商报
- **URL匹配**：`changjiangtimes.com`
- **提取规则**：从 `<div class="c_zw">` 中提取正文
- **实现位置**：`extractArticleContent` 方法

### 3. 新浪财经
- **URL匹配**：`finance.sina.com.cn`
- **提取规则**：从 `<div class="article" id="artibody">` 中提取正文
- **实现位置**：`extractArticleContent` 方法
- **注意**：与新浪新闻（sina.com.cn）的处理逻辑类似，但针对财经频道

### 4. 科创板日报
- **URL匹配**：`stcn.com` 或 `chinastarmarket.cn`
- **提取规则**：从 `<div class="_18p7x" data-testid="article">` 中提取正文
- **实现位置**：`extractArticleContent` 方法

## 📝 修改的文件

- `server/utils/newsAnalysis.js`
  - 在 `extractArticleContent` 方法中添加了4个网站的特殊处理逻辑
  - 位置：在中国经营网处理之后、东方财富网处理之前

## 🔧 实现逻辑

所有特殊处理都遵循相同的模式：

1. **URL检测**：检查URL是否包含特定域名
2. **智能匹配**：使用正则表达式查找目标div标签
3. **嵌套处理**：正确处理嵌套的div标签，避免提取不完整
4. **Script标签过滤**：跳过script标签内的div，避免提取JavaScript代码
5. **内容清理**：清理script、style、noscript和注释标签
6. **长度验证**：如果提取的内容少于100字符，回退到通用提取逻辑

### 2. 预期日志输出

成功提取时应该看到：
```
[extractArticleContent] 检测到XXX URL，使用特殊提取逻辑
[extractArticleContent] ✓ 成功提取XXX正文，长度: XXX字符
```

如果提取失败或内容太短：
```
[extractArticleContent] ⚠️ XXX提取的内容太短（XXX字符），继续使用通用提取逻辑
或
[extractArticleContent] ⚠️ 未找到XXX的XXX div，继续使用通用提取逻辑
```

### 3. 检查提取结果

- 正文内容应该完整提取
- 不应该包含导航、广告等无关内容
- 内容长度应该合理（通常大于100字符）

## ⚠️ 注意事项

1. **URL匹配**：确保URL包含正确的域名，否则不会触发特殊处理
2. **HTML结构变化**：如果网站更新了HTML结构，可能需要调整提取规则
3. **回退机制**：如果特殊提取失败或内容太短，会自动回退到通用提取逻辑
4. **优先级**：特殊处理在通用提取之前执行，优先级更高

## 🔍 故障排查

### 问题1：未检测到网站URL

**检查**：
- 确认URL是否包含正确的域名
- 查看日志中是否有 "检测到XXX URL" 的日志

**解决**：
- 检查URL格式是否正确
- 确认域名匹配规则是否准确

### 问题2：未找到目标div

**检查**：
- 查看日志中是否有 "未找到XXX的XXX div" 的日志
- 检查HTML中是否确实存在目标div

**解决**：
- 检查实际的HTML结构
- 可能需要调整正则表达式匹配规则

### 问题3：提取的内容不完整

**检查**：
- 查看提取的内容长度
- 检查是否有嵌套div导致提取不完整

**解决**：
- 确认HTML结构是否与预期一致
- 检查嵌套div的处理逻辑是否正确

### 问题4：提取的内容包含无关信息

**检查**：
- 查看提取的内容
- 检查清理逻辑是否正确执行

**解决**：
- 可能需要添加更多的清理规则
- 检查script、style等标签是否被正确过滤

## 📊 处理流程

```
提取正文内容
    ↓
检测URL域名
    ↓
匹配到特定网站？
    ↓ 是
使用特殊提取规则
    ↓
查找目标div标签
    ↓
提取div内容（处理嵌套）
    ↓
清理无关标签
    ↓
验证内容长度
    ↓
长度 > 100字符？
    ↓ 是
返回提取的内容
    ↓ 否
回退到通用提取逻辑
```
