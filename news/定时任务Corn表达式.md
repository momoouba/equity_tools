Cron 表达式可视化生成器 开发需求文档
一、开发目标
基于 React 18 + TypeScript + Ant Design 5.x 开发「Cron 表达式可视化配置组件」，实现秒 / 分 / 时 / 日 / 月 / 周 / 年 7 个维度的可视化配置，支持「跳过节假日」过滤逻辑，实时生成 Cron 表达式并计算符合条件的预计执行时间，界面与交互效果需满足设计要求。
二、界面布局要求
1. 整体结构

2. 标签页（7 个维度）
采用 Ant Design Tabs 组件实现，默认选中「秒」标签；
样式：选中标签为 Ant Design 蓝色主题，未选中为默认灰色样式；
切换标签时仅刷新当前维度配置区，保留其他维度已配置数据。
3. 配置区
3.1 基础配置项（每个维度通用）
每个维度包含 4 种配置模式（Radio 单选按钮切换）：
配置模式	控件要求
通配符（*）	展示说明文字（如「每秒执行」），无输入控件；
周期范围	2 个 InputNumber 输入框，标注「从」「到」，支持数值步进 1，限制对应维度边界；
间隔执行	2 个 InputNumber 输入框，标注「从」「每」，支持数值步进 1，限制对应维度边界；
指定具体值	Input 输入框，支持手动输入 / 下拉选择，多个值用逗号分隔（如 1,5,10）；
3.2 跳过节假日开关
位置：配置区顶部，Ant Design Switch 组件，标注文字「跳过节假日」；
默认状态：未勾选；
交互：勾选 / 取消时立即重新计算执行时间预览区数据。
4. 表达式预览区
左侧：Ant Design Table 组件，7 行（秒 / 分 / 时 / 日 / 月 / 周 / 年）×1 列，单元格支持编辑；
右侧：Input 文本框，展示完整 7 位 Cron 表达式（如 */5 * * * * ? *），支持手动编辑；
联动规则：配置区修改→表格 + 文本框实时更新；文本框 / 表格编辑→配置区自动同步。
5. 执行时间预览区
标题：「最近 5 次执行时间」；
内容：展示格式为 YYYY-MM-DD HH:mm:ss 的时间列表，最多显示 5 条；
提示规则：
无有效表达式：显示「请配置有效 Cron 表达式」（Alert 组件）；
数据加载中：显示骨架屏（Skeleton 组件）；
过滤后无数据：显示「暂无符合条件的执行时间（已跳过节假日）」（Empty 组件）；
表达式错误：提示「表达式格式错误，请检查配置」（Message 组件）。
6. 底部操作按钮
按钮顺序：取消（左）、重置（中）、确定（右）；
样式：确定按钮为 Primary 类型，取消 / 重置为默认类型；
尺寸：统一使用 Ant Design 中等尺寸（default）。
三、核心功能要求
1. Cron 表达式生成规则
支持 7 位 Cron 表达式（秒 分 时 日 月 周 年），兼容 Quartz Cron 规范；
周（星期）和日维度互斥：设置日维度非通配符时，周维度自动设为 ?；反之亦然；
各维度数值边界限制：
秒 / 分：0-59；小时：0-23；日：1-31；月：1-12；周：1-7（1 = 周日）；年：1970-2099。
2. 表达式双向同步
配置区修改：实时解析为对应维度表达式，更新表格和完整 Cron 文本框；
文本框 / 表格编辑：失去焦点后自动解析值，同步更新对应维度的配置模式和输入值；
解析失败时：提示「表达式格式错误」，保留原配置不修改。
3. 节假日过滤逻辑
3.1 数据读取
从系统数据库表 holiday_calendar 读取节假日日期（字段：holiday_date，格式：YYYY-MM-DD）；
封装 getHolidayList 函数，支持接口请求获取数据，预留模拟数据适配方案。
3.2 过滤规则
未勾选「跳过节假日」：直接展示 Cron 表达式计算的最近 5 次执行时间；
勾选「跳过节假日」：
生成≥10 条候选执行时间（避免过滤后无数据）；
剔除日期在节假日列表中的时间；
从剩余时间中取最近 5 条展示；
仅对比「年月日」，不限制时分秒。
3.3 容错处理
节假日数据加载失败：自动关闭开关，提示「节假日数据加载失败，暂无法过滤」；
过滤后无有效时间：展示空状态提示，不报错。
4. 操作按钮功能
取消：关闭组件（弹窗场景调用 destroy 方法，页面场景重置为初始状态）；
重置：恢复所有配置为默认值（所有维度设为通配符 *，开关置为未勾选）；
确定：触发 onChange 回调，返回当前完整 Cron 表达式，父组件可通过 props 接收。
四、技术要求
1. 技术栈
框架：React 18（函数式组件 + Hooks：useState/useEffect/useCallback）；
语言：TypeScript（全量类型定义，包括状态、Props、函数返回值）；
UI 库：Ant Design 5.x（核心组件：Tabs/Radio/InputNumber/Table/Button/Switch/Alert/Empty）；
依赖库：
cron-parser：解析 Cron 表达式，计算执行时间；
@types/cron-parser：补充 TypeScript 类型；
dayjs：日期格式化、对比、解析；
axios（可选）：接口请求获取节假日数据。
2. 代码规范
2.1 组件拆分
父组件：CronGenerator（整合所有子组件，管理核心状态）；
子组件：
CronConfig：配置区（含标签页、配置模式、跳过节假日开关）；
CronPreview：表达式预览区（表格 + 文本框）；
CronSchedule：执行时间预览区（含节假日过滤）。
