接口基础信息

接口地址（测试环境 - 内网）：http://114.141.181.181:8000/dofp/v2/ipaas/query/execPers
请求方式：POST
正文格式：application/json; charset=UTF-8
Header 头信息：与 1.12 接口一致
请求报文格式
json
{
    "exec_idtfn_cd":"91310000132204777G"
}

exec_idtfn_cd	机构唯一识别码	String	是	无
