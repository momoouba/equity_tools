## 🔧 港交所 iFinD 配置说明

### 功能说明
- 港交所上市申请数据可通过同花顺 iFinD 接口获取
- **Windows 本地环境**：使用本地 SDK + 用户名/密码登录
- **Linux 生产环境**：使用 HTTP API + refresh_token 获取 access_token

### 架构说明

```
┌─────────────────┐     ┌─────────────────┐
│  Windows 环境   │     │  Linux/Docker   │
├─────────────────┤     ├─────────────────┤
│ iFinDPy SDK     │     │ HTTP API        │
│ THS_iFinDLogin  │     │ get_access_token│
│ THS_DR          │     │ data_pool       │
└────────┬────────┘     └────────┬────────┘
         │                       │
         └───────────┬───────────┘
                     ↓
            ┌─────────────────┐
            │  数据筛选映射    │
            │ 首次申请→递交A1  │
            │ 聆讯日期→通过聆讯│
            └────────┬────────┘
                     ↓
            ┌─────────────────┐
            │   入库去重      │
            └─────────────────┘
```

### 详细配置步骤

#### Windows 开发环境

1. **下载 SDK**
   - 从同花顺官网下载 Windows 版 iFinD 数据接口 SDK
   - 解压到项目根目录 `THSDataInterface_Windows/`

2. **安装 SDK**
   ```bash
   cd THSDataInterface_Windows
   python installiFinDPy.py
   ```

3. **获取 refresh_token**
   - 打开同花顺 iFinD 金融数据终端
   - 登录账号 → 「超级命令」→「工具」→「refresh_token 查询」
   - 复制 refresh_token

4. **配置系统**
   - 进入「系统设置 → 上市数据配置」
   - 开启「启用 iFinD 港交所」
   - 填写用户名、密码或 refresh_token
   - 保存配置

#### Linux/Docker 生产环境

1. **准备 SDK（已在 Dockerfile 中配置）**
   - 无需手动安装，Docker 构建时自动处理
   - 系统依赖：`libidn11`, `libgssapi-krb5-2`, `libldap-2.5-0`

2. **配置 Token**
   - 使用 Windows 环境获取的 refresh_token
   - 在管理后台配置页面填入 Token 字段
   - 系统将自动通过 HTTP API 获取数据

### 状态映射规则

| 日期匹配条件 | 系统状态 | 说明 |
|-------------|---------|------|
| 首次申请日期在区间内 | **递交A1** | 新递表 |
| 通过聆讯日期在区间内 | **通过聆讯** | 已通过聆讯 |
| 上市日期在区间内 | **上市** | 已上市 |
| 状态更新日期在区间内 + 终止状态 | 保持原状态 | 失效/撤回/被拒绝/被发回 |

### 常见错误排查

**错误1：ModuleNotFoundError: No module named 'pandas'**

**错误2：iFinD 登录失败（返回码 -2）**
```bash
# 原因：refresh_token 无效或过期
# 解决：
# 1. 在 Windows iFinD 终端重新获取 refresh_token
# 2. 更新配置页面的 Token
# 3. 注意：Windows 生成的 Token 不能直接在 Linux 本地 SDK 使用，必须使用 HTTP API 方式
```

**错误3：源表行数=0**
```bash
# 可能原因：
# 1. 日期范围内确实无数据
# 2. HTTP API 调用失败
# 3. refresh_token 无效

# 排查步骤：
# 1. 检查日期范围是否正确
# 2. 检查 refresh_token 是否过期
# 3. 手动测试 HTTP API：
sudo docker compose exec app python3 << 'PYEOF'
import urllib.request
import json
refresh_token = "你的token"
url = "https://quantapi.51ifind.com/api/v1/get_access_token"
headers = {"Content-Type": "application/json", "refresh_token": refresh_token}
req = urllib.request.Request(url, headers=headers, method="POST")
try:
    with urllib.request.urlopen(req, timeout=30) as response:
        data = json.loads(response.read().decode('utf-8'))
        print("access_token:", data.get('data', {}).get('access_token', '获取失败'))
except Exception as e:
    print("错误:", e)
PYEOF
```

**错误4：Windows 本地 SDK 登录失败**
```bash
# 原因：iFinD 客户端未启动或 SDK 未安装
# 解决：
# 1. 确保 iFinD 金融数据终端已启动
# 2. 重新运行 SDK 安装脚本
# 3. 检查 Python 路径是否正确
```
