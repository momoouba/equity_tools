## 🔧 港交所 iFinD 配置说明

### 功能说明
- 港交所上市申请数据可通过同花顺 iFinD 接口获取
- Windows 本地环境：使用用户名+密码登录
- Linux 生产环境：使用 Token 登录

### 配置步骤
1. 在「系统设置 → 上市数据配置」中编辑配置
2. 开启「启用 iFinD 港交所」
3. 填写凭证：
   - Windows 环境：用户名 + 密码
   - Linux 环境：Token（从 Windows 端生成后复制）
4. 保存配置

### 验证 iFinD 同步
```bash
# 查看同步日志
sudo docker compose logs app | grep -i ifind

# 手动触发同步测试
# 在前端「上市数据配置」页面点击「同步」按钮
```

## ⚠️ 注意事项

### 1. 必须重新构建镜像

本次更新包含 Python 依赖变更，必须重新构建 Docker 镜像：

```bash
sudo docker compose build --no-cache app
sudo docker compose up -d
```

### 2. iFinD Token 配置

Linux 生产环境需要使用 Token 登录：
- 在 Windows 环境生成 Token
- 在配置页面填入 Token 字段
- 密码字段可留空

### 3. 备份数据（重要）

在生产环境更新前，建议：
```bash
# 备份数据库
sudo docker compose exec mysql mysqldump -u newsapp -p investment_tools > backup_$(date +%Y%m%d_%H%M%S).sql
```
