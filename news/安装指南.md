# 安装指南

## 问题：'npm'不是内部或外部命令

这个错误表示您的系统还没有安装 Node.js。

## 解决方案

### 方法一：安装 Node.js（推荐）

1. **下载 Node.js**
   - 访问 Node.js 官网：https://nodejs.org/
   - 下载 LTS（长期支持）版本（推荐版本 18.x 或 20.x）
   - 选择 Windows Installer (.msi) 64位版本

2. **安装 Node.js**
   - 运行下载的 .msi 安装程序
   - 按照安装向导操作
   - **重要**：确保勾选 "Add to PATH" 选项（通常默认已勾选）
   - 完成安装

3. **验证安装**
   - 关闭当前的 CMD 窗口
   - 重新打开一个新的 CMD 窗口
   - 运行以下命令验证：
   ```cmd
   node --version
   npm --version
   ```
   - 如果显示版本号，说明安装成功

4. **安装项目依赖**
   ```cmd
   cd E:\USER\SUREAL\Desktop\news
   npm run install-all
   ```

### 方法二：使用包管理器安装（可选）

如果您使用 Chocolatey 或 winget：

**使用 Chocolatey：**
```cmd
choco install nodejs
```

**使用 winget：**
```cmd
winget install OpenJS.NodeJS.LTS
```

## 安装 MySQL

1. 访问 https://dev.mysql.com/downloads/ 下载并安装 MySQL Community Server 8.0+
2. 安装过程中记住 root 密码，或创建专用账号
3. 安装完成后，通过以下命令验证：
   ```cmd
   mysql --version
   ```
4. (可选) 创建独立账号与数据库：
   ```sql
   CREATE DATABASE investment_tools CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
   CREATE USER 'investment_user'@'localhost' IDENTIFIED BY 'StrongPassword123!';
   GRANT ALL PRIVILEGES ON investment_tools.* TO 'investment_user'@'localhost';
   FLUSH PRIVILEGES;
   ```

## 配置环境变量

在项目根目录创建 `.env` 文件，内容参照：
```
DB_HOST=localhost
DB_PORT=3306
DB_USER=investment_user
DB_PASSWORD=StrongPassword123!
DB_NAME=investment_tools
```

## 安装完成后

1. **重新打开 CMD**
   - 关闭所有 CMD 窗口
   - 重新打开一个新的 CMD 窗口

2. **进入项目目录**
   ```cmd
   cd E:\USER\SUREAL\Desktop\news
   ```

3. **安装依赖**
   ```cmd
   npm run install-all
   ```

4. **启动项目**
   ```cmd
   npm run dev
   ```

## 常见问题

### Q: 安装后仍然提示找不到 npm？
**A:** 
- 关闭所有 CMD/PowerShell 窗口
- 重新打开新的窗口
- 如果还是不行，检查系统环境变量 PATH 中是否包含 Node.js 的安装路径

### Q: 如何检查 Node.js 是否已安装？
**A:** 在 CMD 中运行：
```cmd
where node
where npm
```
如果显示路径，说明已安装。

### Q: 需要管理员权限吗？
**A:** 安装 Node.js 需要管理员权限，但运行项目通常不需要。
