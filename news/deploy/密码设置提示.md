# 密码设置提示

## ⚠️ 常见错误

### 错误：`Sorry, passwords do not match`

**原因**：两次输入的密码不一致

**解决方法**：
1. 仔细输入密码（密码输入时不会显示字符）
2. 确保两次输入的密码完全相同
3. 注意大小写、空格、特殊字符

### 错误：`passwd: Authentication token manipulation error`

**原因**：
- 密码不匹配
- 密码不符合系统要求（太简单）
- 权限问题

**解决方法**：
1. 确保两次密码输入一致
2. 使用更复杂的密码（至少8位，包含字母和数字）
3. 使用root权限执行

## ✅ 正确的密码设置方法

### 方法1：使用 passwd 命令

```bash
# root用户设置其他用户密码
sudo passwd guofang
```

**操作步骤**：
1. 输入命令后按回车
2. 看到 `New password:` 提示，输入新密码（**不显示字符**）
3. 按回车
4. 看到 `Retype new password:` 提示，**再次输入相同密码**
5. 按回车
6. 如果成功，会显示 `passwd: password updated successfully`

**重要提示**：
- 密码输入时**不会显示任何字符**（包括星号），这是正常的安全行为
- 必须确保两次输入的密码**完全相同**
- 建议先在一个文本编辑器中准备好密码，然后复制粘贴（但要注意，某些终端可能不支持粘贴）

### 方法2：使用脚本创建用户（自动设置密码）

```bash
sudo ./deploy/create-user.sh guofang
```

脚本会使用 `adduser` 命令，会提示设置密码。

## 🔐 密码要求

### Ubuntu/Debian 系统密码要求：

1. **最小长度**：通常至少6-8个字符
2. **复杂度**：建议包含：
   - 大写字母
   - 小写字母
   - 数字
   - 特殊字符（可选）

### 密码示例（符合要求）：

```
MyPass123
Secure2024!
NewsApp@2024
```

### 密码示例（不符合要求）：

```
123456        # 太简单
password      # 太常见
abc           # 太短
```

## 🛠️ 故障排查

### 问题1：密码输入时看不到字符

**这是正常的**！Linux系统为了安全，密码输入时不显示任何字符。

**解决方法**：
- 仔细输入密码
- 可以先用文本编辑器准备好密码，然后手动输入
- 确保两次输入完全一致

### 问题2：总是提示密码不匹配

**可能原因**：
1. 输入时按错了键
2. 键盘布局问题（如中文输入法）
3. 复制粘贴时包含了空格或换行

**解决方法**：
1. 确保使用英文输入法
2. 手动输入，不要复制粘贴
3. 输入时仔细检查，可以先用简单密码测试

### 问题3：密码设置后无法登录

**可能原因**：
1. SSH配置禁用了密码登录
2. 用户账户被锁定
3. 密码输入错误

**解决方法**：

```bash
# 检查SSH配置
sudo grep PasswordAuthentication /etc/ssh/sshd_config

# 如果显示 no，需要启用密码登录或使用SSH密钥
sudo nano /etc/ssh/sshd_config
# 设置 PasswordAuthentication yes
sudo systemctl restart sshd

# 检查用户账户状态
sudo passwd -S guofang
```

## 💡 最佳实践

### 1. 使用强密码

- 至少12个字符
- 包含大小写字母、数字、特殊字符
- 不使用常见单词或个人信息

### 2. 配置SSH密钥（更安全）

设置SSH密钥后，可以禁用密码登录：

```bash
# 配置SSH密钥
ssh-copy-id guofang@服务器IP

# 禁用密码登录
sudo nano /etc/ssh/sshd_config
# 设置 PasswordAuthentication no
sudo systemctl restart sshd
```

### 3. 定期更换密码

```bash
# 用户自己修改密码
passwd

# root修改其他用户密码
sudo passwd guofang
```

## 📋 快速参考

### 设置密码

```bash
# root设置其他用户密码
sudo passwd guofang

# 用户自己修改密码
passwd
```

### 检查密码状态

```bash
# 查看用户密码状态
sudo passwd -S guofang

# 锁定用户账户（禁用密码登录）
sudo passwd -l guofang

# 解锁用户账户
sudo passwd -u guofang
```

### 重置密码

```bash
# 如果忘记密码，root可以重置
sudo passwd guofang
# 输入新密码两次
```
