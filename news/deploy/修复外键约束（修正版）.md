# 修复外键约束（修正版）

## ❌ 错误

```
ERROR 1064 (42000): You have an error in your SQL syntax; check the manual that corresponds to your MySQL server version for the right syntax to use near 'IF EXISTS fk_news_interface_config_app_id'
```

**原因**：MySQL 的 `DROP FOREIGN KEY` 不支持 `IF EXISTS` 语法。

## ✅ 正确的修复方法

### 方法1：先检查后删除（推荐）

```bash
# 修复外键约束（先检查是否存在，再删除）
sudo mysql -u root -p'Mqdqxygyqy!!!klklsys24678' <<EOF
USE investment_tools;

-- 查找外键约束名称
SELECT CONSTRAINT_NAME 
FROM INFORMATION_SCHEMA.KEY_COLUMN_USAGE 
WHERE TABLE_SCHEMA = 'investment_tools' 
AND TABLE_NAME = 'news_interface_config' 
AND REFERENCED_TABLE_NAME = 'applications'
AND COLUMN_NAME = 'app_id';

-- 如果找到外键，删除它（需要手动执行，或使用下面的方法2）
EOF
```

### 方法2：使用错误处理（自动）

```bash
# 修复外键约束（使用错误处理）
sudo mysql -u root -p'Mqdqxygyqy!!!klklsys24678' <<EOF
USE investment_tools;

-- 禁用外键检查
SET FOREIGN_KEY_CHECKS = 0;

-- 尝试删除外键（如果不存在会报错，但会被忽略）
-- 先查找外键名称
SET @fk_name = NULL;
SELECT CONSTRAINT_NAME INTO @fk_name
FROM INFORMATION_SCHEMA.KEY_COLUMN_USAGE 
WHERE TABLE_SCHEMA = 'investment_tools' 
AND TABLE_NAME = 'news_interface_config' 
AND REFERENCED_TABLE_NAME = 'applications'
AND COLUMN_NAME = 'app_id'
LIMIT 1;

-- 如果找到外键，删除它
SET @sql = IF(@fk_name IS NOT NULL, 
    CONCAT('ALTER TABLE news_interface_config DROP FOREIGN KEY ', @fk_name), 
    'SELECT "No foreign key to drop"');
PREPARE stmt FROM @sql;
EXECUTE stmt;
DEALLOCATE PREPARE stmt;

-- 重新添加外键约束
ALTER TABLE news_interface_config 
ADD CONSTRAINT fk_news_interface_config_app_id 
FOREIGN KEY (app_id) REFERENCES applications(id) ON DELETE CASCADE;

-- 重新启用外键检查
SET FOREIGN_KEY_CHECKS = 1;

SELECT '外键约束修复完成' AS result;
EOF
```

## 🚀 推荐的一键修复（方法2）

```bash
# 自动修复外键约束
sudo mysql -u root -p'Mqdqxygyqy!!!klklsys24678' <<EOF
USE investment_tools;
SET FOREIGN_KEY_CHECKS = 0;
SET @fk_name = NULL;
SELECT CONSTRAINT_NAME INTO @fk_name
FROM INFORMATION_SCHEMA.KEY_COLUMN_USAGE 
WHERE TABLE_SCHEMA = 'investment_tools' 
AND TABLE_NAME = 'news_interface_config' 
AND REFERENCED_TABLE_NAME = 'applications'
AND COLUMN_NAME = 'app_id'
LIMIT 1;
SET @sql = IF(@fk_name IS NOT NULL, 
    CONCAT('ALTER TABLE news_interface_config DROP FOREIGN KEY ', @fk_name), 
    'SELECT "No foreign key to drop"');
PREPARE stmt FROM @sql;
EXECUTE stmt;
DEALLOCATE PREPARE stmt;
ALTER TABLE news_interface_config 
ADD CONSTRAINT fk_news_interface_config_app_id 
FOREIGN KEY (app_id) REFERENCES applications(id) ON DELETE CASCADE;
SET FOREIGN_KEY_CHECKS = 1;
SELECT '外键约束修复完成' AS result;
EOF

# 重启应用
pm2 restart newsapp
```
