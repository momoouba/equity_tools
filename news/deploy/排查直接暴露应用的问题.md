# 排查直接暴露应用的问题

## ✅ 当前状态

- Nginx 未安装
- Apache 未安装
- 应用直接监听在 `0.0.0.0:3001`（公网可访问）
- 没有反向代理

## 🔍 问题分析

浏览器访问 `http://news.gf-dsai.com` 返回 500 错误，但：
- 本地 `curl http://localhost:3001/api/auth/login` 成功
- 应用直接暴露在 3001 端口

可能的原因：
1. 浏览器访问的 URL 可能包含端口号（如 `http://news.gf-dsai.com:3001`）
2. 域名解析可能有问题
3. 可能有云负载均衡器或端口转发

## ✅ 排查步骤

### 步骤1：检查域名解析

```bash
# 检查域名解析
nslookup news.gf-dsai.com
dig news.gf-dsai.com +short

# 检查域名是否解析到服务器 IP（119.3.127.211）
```

### 步骤2：测试直接访问应用

```bash
# 使用服务器 IP 测试
curl http://119.3.127.211:3001/api/system/basic-config

# 使用域名测试（如果域名解析正确）
curl http://news.gf-dsai.com:3001/api/system/basic-config
```

### 步骤3：检查浏览器实际访问的 URL

在浏览器中：
1. 打开开发者工具（F12）
2. 查看 Network 标签
3. 尝试登录
4. 查看实际请求的 URL（是否包含端口号）

### 步骤4：检查云服务器配置

如果使用云服务器（华为云等），检查：
1. 安全组是否开放了 3001 端口
2. 是否有负载均衡器配置
3. 是否有端口转发规则

## 🔧 解决方案

### 方案1：使用 Nginx 反向代理（推荐）

```bash
# 安装 Nginx
sudo apt update
sudo apt install nginx -y

# 创建配置文件
sudo nano /etc/nginx/sites-available/news.gf-dsai.com
```

配置文件内容：
```nginx
server {
    listen 80;
    server_name news.gf-dsai.com;

    location /api {
        proxy_pass http://localhost:3001;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
    }

    location / {
        proxy_pass http://localhost:3001;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```

启用配置：
```bash
# 创建符号链接
sudo ln -s /etc/nginx/sites-available/news.gf-dsai.com /etc/nginx/sites-enabled/

# 测试配置
sudo nginx -t

# 重新加载 Nginx
sudo systemctl reload nginx
```

### 方案2：直接使用端口访问

如果不想配置 Nginx，可以：
- 浏览器访问：`http://news.gf-dsai.com:3001`
- 确保云服务器安全组开放了 3001 端口
