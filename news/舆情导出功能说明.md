# 舆情导出功能说明

## 📋 功能概述

为舆情信息页面添加了Excel导出功能，用户可以根据不同时间范围导出舆情数据，支持多种导出选项和智能文件命名。

## 🎯 主要功能

### 1. Tab页签导出按钮

#### 各Tab页签的导出功能
- **昨日舆情**: 直接导出昨日的舆情数据
- **本周舆情**: 直接导出本周的舆情数据
- **上周舆情**: 直接导出上周的舆情数据
- **本月舆情**: 直接导出本月的舆情数据
- **全部舆情**: 弹出选择器，可选择具体导出范围

### 2. 全部舆情的导出选项

当在"全部舆情"tab点击导出时，提供四个选项：
1. **本周舆情** - 本周一至今的舆情信息
2. **本月舆情** - 本月1日至今的舆情信息
3. **上月舆情** - 上个月完整的舆情信息
4. **全部舆情** - 所有舆情信息

### 3. 智能文件命名

#### 文件名格式
- **单日数据**: `20251120舆情信息.xlsx`
- **日期范围**: `20251117-20251121舆情信息.xlsx`
- **月份数据**: `2025年11月舆情信息.xlsx`
- **全部数据**: `全部舆情信息.xlsx`

#### 命名规则
```javascript
// 单日格式：YYYYMMDD
formatDateForFileName(date) // 20251120

// 范围格式：YYYYMMDD-YYYYMMDD  
`${startDate}-${endDate}舆情信息` // 20251117-20251121舆情信息

// 月份格式：YYYY年MM月
`${year}年${month}月舆情信息` // 2025年11月舆情信息
```
