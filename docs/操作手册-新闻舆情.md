# 股权投资小工具锦集 —— 新闻舆情功能操作手册

> 说明：本手册用于软著登记材料中对“新闻舆情”功能的操作说明。实际提交时可将本内容复制到 Word 文档并另存为 `.docx` 格式。

---

## 一、系统概述

“新闻舆情”功能用于集中展示与被投企业、基金相关主体、子基金等对象有关的微信公众号新闻，并支持按企业、基金、时间维度进行筛选、导出和分享，帮助投资管理人员及时掌握舆情动态。

主要功能模块包括：

- 舆情监控对象：维护被投企业及基金相关主体列表
- 舆情信息：浏览、筛选、导出舆情新闻
- 公众号管理：维护需要监控的公众号
- 收件管理：配置舆情邮件接收人及查看发送记录
- 舆情信息分享：生成外部访问链接，对外展示舆情列表

---

## 二、登录与权限说明

1. 在登录页输入账号和密码，点击“登录”进入系统。
2. 登录成功后进入“控制台”主页，在左侧菜单中点击“舆情监控对象”或“舆情信息”进入舆情功能。
3. 权限区分：
   - 管理员（admin）：
     - 可以查看所有用户创建的舆情监控对象和舆情信息
     - 可以管理公众号、收件人配置及邮件发送日志
     - 可以配置并使用舆情信息对外分享链接
   - 普通用户：
     - 仅能查看自己负责的企业与公众号的舆情信息
     - 仅能管理自己名下的收件配置与查看个人邮件记录

## 三、舆情监控对象管理（“舆情监控对象”菜单）

### 3.1 功能入口

1. 在左侧菜单点击“舆情监控对象”。
2. 页面上方为筛选区域，下方为被投企业/基金主体的列表。

### 3.2 新增舆情监控对象

1. 点击右上角“新增”按钮，弹出“企业信息维护”弹窗。
2. 主要字段说明：
   - 项目编号：系统自动生成（示例：P2026031100001）
   - 企业类型：被投企业 / 基金相关主体 / 子基金 / 子基金管理人 / 子基金GP
   - 项目简称：企业或项目简称
   - 被投企业全称：工商登记的企业全称
   - 统一信用代码：企业的统一社会信用代码
   - 企业公众号id：与该企业相关的微信公众号原始 ID，可填写多个，以逗号分隔
   - 企业官网：企业官网链接
   - 关联基金 / 关联子基金：与该企业相关的基金或子基金名称
   - 退出状态：未退出 / 部分退出 / 完全退出 / 继续观察 / 不再观察 / 已上市
3. 填写完成后点击“保存”，系统校验必填项，通过后记录保存并回到列表。

### 3.3 导入 / 导出功能

1. 批量导入：
   - 点击“批量导入”按钮，下载系统提供的模板文件。
   - 按模板填写企业信息，完成后在弹窗中选择文件上传。
   - 上传完成后系统自动校验并导入，导入结果在弹窗中显示成功和失败条数。
2. 导出：
   - 可在筛选条件设置后点击“导出”按钮，导出当前筛选结果对应的所有企业数据为 Excel 文件。

### 3.4 筛选与搜索

1. 顶部“关键词”输入框支持对以下字段模糊搜索：
   - 项目编号、项目简称、被投企业全称、统一信用代码、公众号 ID、官网、退出状态等。
2. 管理员可通过“筛选用户”下拉框查看指定用户维护的企业列表。
3. 通过 Tab 页“全部 / 被投企业 / 基金相关主体 / 子基金 / 子基金管理人及GP”快速按企业类型过滤。

## 四、舆情信息列表（“舆情信息”菜单）

### 4.1 功能入口与页面结构

1. 在左侧菜单点击“舆情信息”进入舆情列表页面。
2. 页面上方为统计区和时间范围 Tab；中间为工具栏与过滤条件；下方为舆情列表与分页。

主要字段：

- 企业类型
- 被投企业简称 / 全称
- 基金相关主体
- 关联子基金
- 关键词
- 发布时间
- 标题
- 新闻摘要
- 文章链接
- 公众号名称
- 微信账号

### 4.2 多标签模糊搜索（管理员和普通用户通用）

1. 在右上角“搜索输入框”中输入关键词，例如“动脉网”，按回车键。
2. 输入的关键词会生成一个搜索标签（Tag），标签后带有“×”删除按钮。
3. 可继续输入第二个关键词，例如“投资”，再次回车生成第二个标签。
4. 当前搜索逻辑为 **“或”关系（OR）**：
   - 只要任意一个标签匹配以下任意字段，即会返回该条新闻：
     - 标题（title）
     - 新闻摘要（news_abstract）
     - 被投企业全称（enterprise_full_name）
     - 基金相关主体（fund）
     - 关联子基金（sub_fund）
     - 被投企业简称（enterprise_abbreviation）
     - 公众号名称（account_name）
     - 微信账号（wechat_account）
5. 删除标签：
   - 点击标签右侧的“×”可删除该条件，列表自动根据剩余标签重新筛选。
6. 清空所有标签：
   - 点击“清空”按钮，一次性移除全部搜索标签并恢复无搜索条件的列表。

### 4.3 时间范围和企业类型筛选

1. 时间范围 Tab：
   - 昨日舆情、本周舆情、上周舆情、本月舆情、全部舆情
   - 点击不同 Tab，会按照发布时间或创建时间自动过滤列表。
2. 企业类型筛选按钮：
   - 企业相关 / 基金相关主体 / 子基金 / 全部
   - 用于限制列表中出现的企业类型，便于按维度查看舆情。

### 4.4 勾选与 AI 分析 / 清理无效关联（管理员）

1. 勾选列表左侧复选框，可选择一条或多条新闻。
2. 点击“AI分析”按钮，对选中的新闻重新进行 AI 摘要或情绪分析。
3. 点击“清理无效关联”按钮，系统会检查选中新闻的企业关联字段，如果对应企业不在被投企业库中，则清空相关字段，保持数据一致性。

### 4.5 导出舆情信息

1. 在工具栏点击“导出”按钮：
   - 当前时间范围为“全部舆情”时，会弹出导出范围选择弹窗，可选择本周、本月、上月或全部舆情。
   - 非“全部舆情”时，直接导出当前时间范围与筛选条件下的全部列表数据。
2. 系统生成 Excel 文件并自动下载，包含主要舆情字段，方便后续归档或汇报。

## 五、公众号管理（“公众号管理”子页签）

### 5.1 功能入口

1. 在“舆情信息”页面顶部的子页签中，切换到“公众号管理”。
2. 页面展示当前系统维护的微信公众号列表。

### 5.2 新增 / 编辑公众号

1. 点击“新增”按钮，填写以下字段：
   - 公众号名称
   - 微信账号
   - 公众号原始 ID
   - 备注信息
2. 点击“保存”后，系统会在后续舆情同步任务中根据这些公众号抓取和匹配新闻。
3. 编辑已有公众号：
   - 在列表中点击“编辑”按钮，修改需要调整的字段并保存。

## 六、收件管理与邮件日志（“收件管理”子页签）

### 6.1 收件管理

1. 在“舆情信息”页面顶部的子页签中选择“收件管理”。
2. 功能包括：
   - 配置个人或团队的舆情邮件接收人（邮箱地址）
   - 设置接收频率（每日、每周、每月）和发送时间
3. 保存配置后，系统会按照设定的频率和时间，自动整理舆情信息并通过邮件发送给收件人。

### 6.2 收发记录与邮件日志（普通用户）

1. 在“收件管理”模块中，切换到“收发记录”或“邮件日志” Tab。
2. 可查询：
   - 历史舆情邮件发送记录
   - 每封邮件中包含的舆情条数和发送时间
3. 便于确认邮件是否正常下发以及排查异常。

## 七、舆情分享功能（公共链接）

### 7.1 管理员在后台开启分享

1. 进入“舆情信息”页面右上角，点击“发布”按钮。
2. 弹出“公共链接分享”配置窗口，内容包括：
   - 是否开启分享：勾选后系统生成可访问的公共链接
   - 是否设置有效期：选择截止日期时间，过期后外部访问将被禁止
   - 是否启用访问密码：可自动生成随机密码，或手动设置密码
3. 点击“创建链接”或“更新链接”按钮后，系统返回分享链接及密码，并自动复制到剪贴板。

### 7.2 外部访问者查看舆情信息

1. 管理员将链接（及密码，如有）发送给需要查看舆情的外部人员。
2. 外部人员在浏览器中打开链接：
   - 若设置了密码，系统会先弹出密码输入框，验证通过后进入舆情列表页面。
3. 外部页面展示内容：
   - 与后台“舆情信息”页面基本一致，包括时间范围 Tab、企业类型过滤、舆情列表和分页。
4. 外部页面支持与后台相同的多标签模糊搜索能力：
   - 可按标题、新闻摘要、被投企业全称、基金相关主体、关联子基金、被投企业简称、公众号名称、微信账号等字段进行 OR 关系查询。

## 八、常见操作示例

### 示例 1：查看昨日与某被投企业相关的全部舆情

1. 打开“舆情信息”页面。
2. 选择时间范围 Tab“昨日舆情”。
3. 在搜索输入框中输入该企业简称或全称，按回车生成标签。
4. 列表中会展示昨日与该企业相关的所有微信新闻。

### 示例 2：筛选“基金相关主体”下的特定关键词舆情

1. 在“舆情信息”页面点击企业类型筛选中的“基金相关主体”。
2. 在搜索输入框中依次输入多个关键词（例如“募资”“清算”等），每输入一个回车生成标签。
3. 列表将展示标题或摘要等字段中包含任一关键词的基金主体相关舆情。

### 示例 3：生成并发送舆情周报分享链接

1. 管理员在“舆情信息”页面选择时间范围为“本周舆情”。
2. 点击“发布”按钮，开启“公共链接分享”，并勾选“密码保护”。
3. 点击“创建链接”，系统生成分享链接和访问密码，并复制到剪贴板。
4. 管理员将链接和密码发送给投委会或管理层，用于在线查看本周舆情。

## 九、注意事项

1. 搜索支持中英文与模糊匹配，建议尽量使用企业简称、全称或公众号名称中的关键字。
2. 若某条新闻的企业关联字段为空，可能是企业名称在被投企业库中不存在或已被清理为无效关联。
3. 导出功能会按照当前筛选条件导出全部数据，请在导出前确认时间范围和过滤条件是否正确。
4. 管理员修改“舆情监控对象”或“公众号管理”中的数据后，新配置会在下一次舆情同步任务时生效。
